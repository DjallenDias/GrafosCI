Legenda

                
    
                    
  • Cada coluna é referente a um período
    • 
                    
  • Vermelho - Matéria não pode ser feita
    • 
                    
  • Azul - Matéria pode ser feita
    • 
                    
  • Verde - Matéria concluída
    • 
                

                
Dicas

                
    
                    
  • É Recomendado o uso através do computador.
    • 
                    
  • Use o botão de rolagem para dar zoom e ter uma melhor visualização.
    • 
                    
  • É possível marcar disciplinas mesmo sem ter os pré-requisitos necessários.
    • 
                    
  • É possível reordenar as disciplinas no "eixo" do período, arrastando para cima ou para baixo.
    • 
                    
  • Ao colocar o mouse em cima de uma disciplina, é possivel ver o nome e código.
    • 
                    
  • Ao clicar e/ou arrastar uma disciplina, as arestas que incidem ascendem e são destacadas mostrando seus respectivos pré-requisitos.
